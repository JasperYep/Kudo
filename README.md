# Kudo

[English](./README.md) | [简体中文](./README.zh-CN.md)

A minimalist todo app with calm, restrained gamification.

## Why Kudo

Most todo apps are good at collecting tasks.  
Kudo is built to help you *move*.

It brings tasks and rewards into one simple loop.

You do the work.  
You earn the coins.  
You buy the reward.

Simple loop. Clear feedback.

## What You Can Do

- Keep tasks in one ordered queue
- Add tasks quickly with a pull-down composer
- Time tasks when the value is unclear, or set a fixed coin value manually
- Import plain-text or Markdown task lists into Tasks
- Earn coins as you complete tasks, with rewards shaped by recent effort
- Spend coins in your personal reward store
- Review your actions in the log and keep lightweight notes
- Keep your data local, with backup, restore, and task import support

## What Makes Kudo Different

### Minimal by default

Kudo stays visually quiet.  
No heavy dashboards, no noisy gamification, no clutter for the sake of “engagement”.

### Gamification with restraint

Coins and rewards are not decoration.  
They are a small system for making effort feel tangible.

### Tasks with shape

Kudo keeps the main task flow direct.

- **Tasks** live in one queue you can reorder, with fixed or timer-based coin values
- **Store** holds personal rewards
- **Log** records completed work and spending

### Private and local

Your data stays on your device.  
Backup and restore work through the system file manager.  
Task import accepts simple pasted text.

## Philosophy

Kudo is built around a simple idea:

> productivity should feel calm, physical, and earned.

You pull down to add.  
Items fall into place.  
You finish work, and the system answers back.

The goal is not to pressure you.  
The goal is to make progress feel real.

## Install

You do not need Google Play.

1. Open the GitHub Releases page
2. Download the latest `Kudo-<version>.apk`
3. Allow installs from your browser or file manager if Android asks
4. Open the APK and install it

Releases: `https://github.com/JasperYep/Kudo/releases`
