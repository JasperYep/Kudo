# Kudo

[English](./README.md) | [简体中文](./README.zh-CN.md)

一个极简、带有游戏化反馈的 Todo App。

## 为什么用 Kudo

很多待办应用擅长“收集任务”。  
Kudo 更在意“推动你完成”。

它把任务和奖励放进同一套简单系统里：

你完成事情。  
你获得金币。  
你兑换奖励。

闭环清晰，反馈直接。

## 你可以做什么

- 在一个有序 Tasks 队列中管理待办
- 从顶部下划快速添加任务
- 不确定任务价值时可以计时，也可以手动设定固定金币值
- 将纯文本或 Markdown 任务列表导入到 Tasks
- 完成任务后获得金币，奖励会随近期投入变化
- 在奖励商店中消费金币，兑换喜欢的奖励
- 在日志中回看行动轨迹，也可以保存轻量笔记
- 数据完全保留在本地，并支持备份、恢复和任务导入

## Kudo 的特点

### 默认就是极简

Kudo 尽量保持安静。  
没有厚重的仪表盘，没有喧闹的装饰，也没有为了“上瘾”而堆出来的刺激。

### 克制的游戏化

金币和奖励不是噱头。  
它们只是让“努力”变得更可见、更有触感的一层反馈。

### 任务有结构

Kudo 保持主任务流直接清晰。

- **Tasks**：一个可排序的待办队列，支持固定金币值或按计时折算
- **Store**：个人奖励商店
- **Log**：记录完成与消费

### 本地优先

你的数据主要保留在设备本地。  
备份和恢复通过系统文件管理器完成。  
任务导入支持直接粘贴简单文本。

## 哲学

Kudo 想表达的是一件很简单的事：

> 效率不该是紧张和堆积，而应该是平静、明确、可感知的推进。

你从顶部下划添加。  
项目落入自己的位置。  
你完成动作，系统给出回应。

它不是为了逼你更焦虑。  
而是为了让进步更真实。

## 安装

不需要 Google Play。

1. 打开 GitHub Releases 页面
2. 下载最新的 `Kudo-<版本号>.apk`
3. 如果 Android 提示，请允许浏览器或文件管理器安装未知应用
4. 打开 APK 并安装

发布页：`https://github.com/JasperYep/Kudo/releases`
